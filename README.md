# zigbee2mqtt-wz-spi
Advanced Zigbee2MQTT external converter for Skydance WZ-SPI (Tuya). Full control of brightness, color, saturation, and effects (including custom 4-color scenes). Supports chip selection (SK6812, WS2811, WS2814), speed, pixel count, and technical settings. Optimized for high-density COB and standard digital strips.
